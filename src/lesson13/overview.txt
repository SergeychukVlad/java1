S13 Заключение

1. Множества, HashSet.
                - множество состоит из уникальных элементов
                - все множества реализуют интерфейс Set
                - множества позволяют добавлять, удалять и проверять наличие элементов
                - Множество HashSet реализовано на основе хеш-таблицы
                - порядок элементов в HashSet не определён
                - проверка уникальности, при добавлении элементов в HashSet, происходит с использованием хеш-таблицы,
                  поэтому окончательное сравнение - методом equals - вызывается только в крайнем случае
                - поиск и удаление элементов (по самому элементу) осуществляется в множествах быстро
                - последовательный доступ к элементам множества по индексу невозможен, но возможен через итератор

2. LinkedHashSet, Проверка типа.
                - конструкторы множеств позволяют создать пустое множество или множество на основе элементов любой коллекции
                - LinkedHashSet сохраняет порядок элементов
                - методы для объединения и пересечения множеств удобно писать с помощью методов addAll и retainAll
                - в методе equals лучше использовать getClass, а не instanceof
                - множества надёжнее создавать на основе неизменяемых (immutable) объектов

3. Интерфейсы NavigableSet, SortedSet. Класс TreeSet.
                - в множестве TreeSet элементы сортируются автоматически
                - для TreeSet необходим компаратор:
                            либо реализованный интерфейс Comparable для элементов
                            либо явно указанный в конструкторе (данный компаратор является приоритетным)
                - для сравнения элементов на уникальность в TreeSet используется не equals, как в HashSet и LinkedHashSet,
                  а compareTo или compare
                - интерфейс NavigableSet дополняет SortedSet оригинальными методами поиска в множестве,
                  а также обратным итератором
                - среда IntelliJ Idea помогает решать проблемы при помощи нажатия Alt+Enter
