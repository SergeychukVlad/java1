S12 Заключение

1. Коллекции, ArrayList.

                в коллекциях хранятся только объекты
                у интерфейса Collection три потомка:
                            интерфейс List
                            интерфейс Set
                            интерфейс Queue
                ArrayList это динамический массив, т.е. массив с возможностью удаления и добавления элементов
                интерфейс Collection даёт базовые методы для работы с коллекциями
                интерфейс List дополняет его методами для работы с элементами по индексу, а также возможностью сортировки
                класс Collections содержит полезные статические функции для работы с коллекциями (в том числе max, min, sort)


2. Устройство списков

                список ArrayList организован на основе массива, а LinkedList представляет из себя классический связанный двунаправленный список
                в большинстве случаев в качестве списка лучше использовать динамический массив: ArrayList
                LinkedList следует использовать, если подавляющее большинство операций это:
                            добавление элементов в начало списка
                            удаление элементов из начала списка
                LinkedList удобно использовать в качестве стека
                при использовании LinkedList следует избегать вызова методов, связанных с доступом по индексу в середину списка: add(n, E), get(n), set(n, E), remove(n) и т.д.


3. Итераторы.

                В ArrayList и LinkedList присутствует два типа итератора:
                            стандартный для всех коллекций (реализует интерфейс Iterator)
                            с расширенными возможностями для списков (реализует интерфейс ListIterator)
                Цикл for-each реализован через стандартный итератор
                ListIterator позволяет добавлять, удалять и менять элементы списка
                ListIterator позволяет двигаться по списку в обе стороны
                При использовании LinkedList лучше избегать вызова методов, связанных с доступом по индексу. Вместо этого следует использовать цикл for-each или итератор listIterator()
