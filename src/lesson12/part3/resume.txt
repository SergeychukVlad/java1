ArrayList / LinkedList используют два типа итераторов:
- стандартный для всех типов коллекций (интерфейс Iterator)
- с расшмренными возможностями для списков (интерфейс ListIterator)

Цикл for-each реализован через стандартный итератор

ListIterator позволяет добавлять, удалять, менять элементы списка

ListIterator позволяет двигаться по списку в обе стороны

При использовании LinkedList надо избегать вызова методов, связанных с доступом по индексу.
Вместо этого следует использовать цикл for-each или итератор ListIterator
